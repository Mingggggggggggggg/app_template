# Flutter Audit App Template

Diese Repository enthält ein robustes, modulares Flutter-Template für Audit- und Inspektions-Apps. Es ist darauf ausgelegt, Prüfprozesse digital auf den mobilen Endgeräten zu vereinfachen.
## Features
  - Dynamische Animationen
  - Offline First: Zur Erstellung der Audits ist keine Internetverbindung erforderlich. Die Audits werden lokal gespeichert.
  - Ähnlich zur Jupyter Notebook können hier in gewünschter Reihenfolge Textblöcke oder Bilder hinzugefügt werden.
  - Dynamische Tags
  - UI Anpassung (Farbe, Lightmode, Darkmode)


## Geplant
  - Einbindung/Synchronisation an Cloudanbieter (pcloud Integration)
  - CRUD für Audits
  - Einbindung der Handykamera
  - PDF Export

## Architektur & Technologien
  - Framework: Flutter
  - SQFlite für die Offline Datenhaltung
  - Custom Widgets
